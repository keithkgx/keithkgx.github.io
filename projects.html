UID Extractor

At my internship, I was receiving and opening hundreds of trade-confirmation emails a day, each with a PDF attachment, to extract a single unique ID into Excel.

To automate this process, I wrote a Python script that:
1. Scans unread Outlook emails in a target folder
2. Opens and parses each attached PDF
3. Extracts its unique trade ID
4. Appends that ID into a centralized Excel log.

Now what took an hour could be done in minutes. Unfortunately I’m not able to share more as this project was done during my internship term.

ID Validation Tool

At my internship, trades are done within the Intercontinental Exchange (ICE). Manually keying ICE trades into our back-end system inevitably led to missing or mistyped IDs.

Using Excel VBA I built a validation tool that cross-checks each ICE trade ID against our internal database, flags any IDs that aren't found, and presents an error report. This ultimately led to a lower error rate.
